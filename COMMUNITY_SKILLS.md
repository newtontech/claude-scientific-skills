## 📋 完整 Skills 列表

### 1. 开发工具类

#### coding-agent
- **描述**: 代码代理工具，支持 Codex、Claude Code 等 AI 编程助手
- **用途**: 将编码任务委托给专门的 AI 编程代理
- **路径**: `community-skills/coding-agent/`

#### docker-essentials
- **描述**: Docker 容器管理基础命令和工作流
- **用途**: 容器操作、镜像管理、调试
- **路径**: `community-skills/docker-essentials/`

#### ralph-loop
- **描述**: AI 代理循环工作流，支持自动化代码生成
- **用途**: 创建自运行的 AI 代码生成循环
- **路径**: `community-skills/ralph-loop/`

#### mcp-builder
- **描述**: MCP (Model Context Protocol) 服务器构建指南
- **用途**: 构建与外部服务集成的 MCP 服务器
- **路径**: `community-skills/mcp-builder/`

#### mcporter
- **描述**: MCP 服务器管理和调用工具
- **用途**: 列出、配置、调用 MCP 服务器
- **路径**: `community-skills/mcporter/`

#### webapp-testing
- **描述**: Web 应用测试工具，基于 Playwright
- **用途**: 验证前端功能、调试 UI、截图
- **路径**: `community-skills/webapp-testing/`

#### acp-orchestrator
- **描述**: ACP (Agent Control Protocol) 工作流编排器
- **用途**: 统一管理 codex、claude、opencode 等工具
- **路径**: `community-skills/acp-orchestrator/`
